gamelist.csv - ФАЙЛ ФОРМАТА csv для списка игр

RomPath, - полный или относительный адресс до Rom-фала
Console, - типо консоли для сортировки в листе (GEN, SMS, NES, SNES)
Name, - отображаемое в листе имя
Publisher, - паблишер (не важно)
CmdFS, - команда для запуска в полноэкранном режиме. %f - подставляет RomPath, %name - подставляет имя из RomPath без расширения 
CmdWin, - так же, что и в CmdFS, но не активно
Cover, - обложка (не важно)
ScreenShots - путь до скриншотов, через | (не важно)

config.ini  - INI файл с настройками

fullscreen - запуск приложения в полножкранном, по умолчанию False
gamelist - адресс до геймлиста, если он отличен от стандартного
favorites - адресс до геймлиста с фаворитами (пока не работает)
sdcard - адресс начальной диреектории для пункта SDCARD
